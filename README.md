## How to Use

### Development

- Start the project: `npm run dev`

### Linting

- Apply ESLint: `npm run lint`

### Type Checking

- Run TypeScript: `npm run typecheck`

### Build

- Build the project: `npm run build`
- Preview the result: `npm run preview`
